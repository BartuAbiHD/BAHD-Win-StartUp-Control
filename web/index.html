Help & Documentation

How to use StartUp Control.

Genel Kullanım

StartUp Control, Windows işletim sisteminizde başlangıçta otomatik olarak çalışan programları ve arka plan hizmetlerini (services) kontrol etmenizi sağlar.

  • Bir programın otomatik başlamasını engelleyebilir veya tekrar açabilirsiniz.
  • Windows Görev Yöneticisi üzerinden devre dışı bırakılmış kayıtlar güvenli şekilde gösterilir.
  • Sistem servislerini durdurabilir veya başlatabilirsiniz. (Yönetici yetkisi gerektirebilir)
